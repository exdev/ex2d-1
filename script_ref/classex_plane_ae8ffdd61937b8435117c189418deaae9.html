virtual void exPlane.OnEnable ( ) [protected, virtual]

OnEnable functoin inherit from MonoBehaviour, When exPlane.enabled set to true, this function will be invoked, exPlane will enable the renderer if they exist.

Note:
if you inherit from exPlane, and implement your own Awake function, you need to override this and call base.OnEnable() in your OnEnable block.

Reimplemented in exSpriteBorder, exSprite, exSoftClip, and exSpriteBase.
