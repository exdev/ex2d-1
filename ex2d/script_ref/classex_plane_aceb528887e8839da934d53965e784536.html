void exPlane.OnDisable ( ) [protected]

OnDisable functoin inherit from MonoBehaviour, When exPlane.enabled set to false, this function will be invoked, exPlane will disable the renderer if they exist.

Note:
if you inherit from exPlane, and implement your own Awake function, you need to override this and call base.OnDisable() in your OnDisable block.

Reimplemented in exSpriteBorder, exSprite, exClipping, exSoftClip, and exSpriteBase.
