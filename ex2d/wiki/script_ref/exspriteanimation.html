exSpriteAnimation

Variables Description
animations The sprite animation clips
defaultAnimation Default animation of the SpriteAnimation
playAutomatically Play default animation when the Game start if it is true
Functions Description
Play(string) Play sprite animation by name
Stop() Stop the playing sprite animation
Pause() Pause the playing sprite animation
Resume() Resume the paused sprite animation
GetAnimation(string) Get sprite animation state by name
GetCurFrameInfo() Get current sprite animation frameinfo
