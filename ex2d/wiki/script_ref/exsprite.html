Variables Description
trimTexture Trim the sprite texture, only affect when sprite texture is not in Atlas
trimRect The rect of the trimmed texture, only affect when sprite texture is not in Atlas
color The color of the sprite
customSize If custom size by self
width The width of the sprite, can be seted when customSize is true
height The height of the sprite, can be seted when customSize is true
atlas The reference atlas, can be seted through SetSprite function
index The index of the texture in atlas, can be seted through SetSprite function
Functions Description
Clear () Clear the sprite, set the reference atlas to null
SetSprite ( Atlas, int ) Set the sprite texture by atlas and index

Show/Hide the sprite

You can control if show or hide the sprite by 'enabled' variable. 

exSprite mySprite = GetComponent<exSprite>();
...
mySprite.enabled = false; // hide the sprite
...
mySprite.enabled = true; // show the sprite
...
