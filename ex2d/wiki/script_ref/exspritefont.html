exSpriteFont

Variables Description
fontInfo The referenced BitmapFont asset
text The text rendered in the SpriteFont
useMultiline Use multiple or single line text in SpriteFont
useKerning Use kerning if we have kerning information in the BitmapFont asset
topColor The top color for gradient
botColor The bottom color for gradient
useOutline Use outline effect render the SpriteFont
outlineWidth The width of the outline effect. only available when useOutline is true
useOutline Use outline effect render the SpriteFont
useShadow Use shadow effect render the SpriteFont
shadowBias The bias of the shadow. only available when useShadow is true
shadowColor The color of the shadow. only available when useShadow is true
Functions Description
Clear() Clear the text in sprite font, set the font info to null

Show/Hide the sprite font

You can control if show or hide the sprite font by enabled variable. 

exSpriteFont myFont = GetComponent<exSpriteFont>();
...
myFont.enabled = false; // hide the sprite font
...
myFont.enabled = true; // show the sprite font
...
