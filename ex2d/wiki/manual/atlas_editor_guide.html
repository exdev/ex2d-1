Atlas Editor Guide

Overview

The Atlas Editor in ex2D allows you create and modify Atlas Assets directly inside Unity3D.

Create New Atlas

Select a folder or file in Project Window. Right click the selected folder/file to popup the Menu. Choose Create → ex2D Atlas Info.

You will see a wizard popup help you create an Atlas:

Option Explanation
Saved Path The saved directory path
Asset Name The saved name of the new asset
Width The width of the atlas texture
Height The height of the atlas texture

Note: Don't worry about the width/height of the atlas texture, you can change them later.

Note: It is highly recommended to use a proper name when creating new Atlas. Because once the Atlas created, it will generate 4 different assets — Atlas Texture, Atlas Material, Atlas EditorInfo and the Atlas itself. A good name can save your time on renaming all of them later. }}

When you click the Create button, Atlas Editor will generate 4 assets: Atlas Texture, Atlas Material, Atlas EditorInfo and Atlas:

Open Atlas Editor

The Atlas Editor can be opened from the ex2D→Editor→Atlas Editor menu.

You can also open the atlas editor by select an Atlas Info you created, and click the Edit… button in the Inspector.

Import Textures to Atlas

Select/Multi-Select Textures in Project Window, drag and drop them on the Atlas View Region in Atlas Editor Window. The Atlas Editor will add the textures to Atlas Asset.

After import, you will see the texture all overlap together. Atlas Editor provides automatically layout method to handle new Imports. By press Apply button in the left of Atlas View Region, the Editor will layout all textures.

Save Changes

When you finish your layout in Atlas Editor, you can save your changes by press Build or press Command/Ctrl + s.

The build process will convert the layout of the textures into one Atlas Texture. After build, the Atlas Editor will search the project and find those Prefab and GameObject reference the built Atlas, update their texture index information.
