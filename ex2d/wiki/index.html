Welcome to ex2D for Unity Documentation Wiki!

ex2D is a Unity extension providing 2D sprite solution. It has the best Atlas Editor and Sprite Animation Editor in the business, and is highly optimized for mobile devices.

This documentation wiki will walk you through in a few easy steps, to help you start rocking Sprites in Unity! You will also find runtime scripting reference for all of ex2D classes.
